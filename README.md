# Mason Plugins Install

Install Mason LSP plugins with `:MasonInstall`.

For C/C++:
```
    ◍ cmake-language-server cmake
    ◍ clang-format 
    ◍ clangd 
    ◍ cpplint
```

For Python:
```
    ◍ black
    ◍ pyright 
    ◍ flake8 
```

Others:
```
    ◍ r-languageserver
    ◍ lua-language-server
    ◍ bash-language-server
    ◍ json-lsp
    ◍ texlab 
    ◍ yaml-language-server
```
